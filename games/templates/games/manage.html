Management

As an umpire you will be able to configure your game from this page, as well as viewing the games of previous umpires. MORE INFORMATION COMING SOON!

  1. Start by creating a group to contain all the elements used for the game.
  2. Create a game for the group, give it a title, description, and timing.
  3. Create the Primary Config for the default rules used throughout your game.
  4. Create the Flags used in your game to represent teams, status effects, etc.
  5. Create the Configs for the special rules in your events.
  6. Package these Configs into Events with offset timings and durations.
  7. Package the Primary Config and Events to a Script, and attatch this to your game.
  8. Congratulations, you're ready to publish your game for sign-ups!
